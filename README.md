# IPRO-VAULT-SERVER

## Installing

Requirements:
Go >=1.4


*NOTE: Make sure your $GOPATH is set*

``
go get -u github.com/skrzepto/IPRO-VAULT-SERVER
``

## Creating a binary

```
cd $GOPATH/src/github.com/skrzepto/IPRO-VAULT-SERVER/
go build
IPRO-VAULT-SERVER
```

now go to localhost:8082

## Running test suite
``
cd $GOPATH/src/github.com/skrzepto/IPRO-VAULT-SERVER/
go test
``
